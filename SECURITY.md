# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Edictum, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Email: security@edictum.ai

We will acknowledge receipt within 48 hours and provide a timeline for a fix.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.x     | Yes       |

## Security Testing

Every security boundary has adversarial bypass tests. See `CLAUDE.md` for the full security review checklist and negative security test requirement.
